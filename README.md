# Free Knots Classifier

Программный комплекс для классификации свободных узлов малой сложности (до 6 хорд).

## Описание

Свободные узлы — объекты комбинаторной топологии, возникающие из виртуальных узлов после забывания информации о стрелках и знаках перекрестков. Комплекс позволяет:

- Генерировать все уникальные хордовые диаграммы для заданного числа хорд
- Применять движения Рейдемейстера (Ω₁, Ω₂, Ω₃) в уменьшающем, неуменьшающем и увеличивающем вариантах
- Анализировать зацепленность, четность и несократимость хорд
- Вычислять инварианты: скобка четности, z-инвариант Гибсона
- Визуализировать хордовые диаграммы и процессы упрощения
- Классифицировать свободные узлы

## Результаты

| Число хорд | Уникальных диаграмм | Нетривиальных узлов |
|------------|---------------------|---------------------|
| 0-4        | 1, 1, 2, 5, 17      | 0                   |
| 5          | 79                  | 2                   |
| 6          | 554                 | 22 + 3(?)           |
